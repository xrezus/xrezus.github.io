Telegram Microservice Template (Go)

I developed a microservice template for quickly building backend applications for Telegram using Go. This template includes the following key features:

  • Authentication: Implemented via Telegram WebApp using InitData.
  • Ready-made Framework: Includes an HTTP API with Fiber v3, JWT authorization, and middleware (RequestID, Auth).
  • Database Support: Works with PostgreSQL and Redis for data storage and session caching.
  • Graceful Shutdown: Ensures proper service termination.
  • Logging: Implemented through a custom logger.
  • Clean Architecture: Organized structure based on repositories, services, and handlers.
